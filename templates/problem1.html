Ash's Sequence

Brock gives Ash an array A1,A2...AN.
He asks him to find an integer M between P and Q(both inclusive), such that, min {|Ai-M|, 1 ≤ i ≤ N} is maximised.
If there are multiple solutions, print the smallest one.

Input

The first line contains N.
The next line contains space separated N integers, and denote the array A.
The third line contains two space separated integers denoting P and Q.

Output

For each test case output the respective answer in a separate line.

Small Dataset (30 points)

Large Dataset (70 points)

1 ≤ N ≤ 102 1 ≤ Ai ≤ 109 1 ≤ P ≤ Q ≤ 109

Sample Input

3
5 8 14
4 9

Sample Output

4
