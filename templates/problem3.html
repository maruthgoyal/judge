Poke Network

People connect with each other in a Poke network.
A connection between Person I and Person J is represented as M I J
When two people belonging to different communities connect, the net effect is the merger of both communities which I and J belong to.

At the beginning, there are N people representing N communities. Suppose person 1 and 2 are connected and later 2 and 3 connected, then 1,2 and 3 will belong to the same community.

There are 2 tpes of queries: 1. M I J ==> Communities containing person I and J merged (if they belong to different communities) 2. Q I ==> Print the size of the community to which person I

Input

The first line of the input will contain integers N and Q, i.e the number of people and the number of queries.
The next Q lines will contain the queries.

Output

The output of the queries.

Large Data set (100 points)

1 ≤ N ≤ 105
1 ≤ Q ≤ 2 x 105

Sample Input

3 6
Q 1
M 1 2
Q 2
M 2 3
Q 3
Q 2

Sample Output

1
2
3
3
